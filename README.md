# 🚀 Employee Management System (Payroll Pro)

A modern, high-performance web application designed to manage employee data, track budgets, and calculate payroll with real-time analytics. Built for HR teams and startup founders who value efficiency and clean UI.

## ✨ Key Features

* **Employee Lifecycle:** Easily add, track, and manage team members.
* **Real-time Financials:** Automatic calculation of Gross and Net budgets.
* **Bonus System:** One-click $500 bonus distribution.
* **Modern Analytics:** Visual progress bars for budget utilization tracking.
* **Responsive UI:** Clean, professional design optimized for high-resolution monitors.

## 🛠️ Tech Stack

* **Frontend:** React (TypeScript)
* **Build Tool:** Vite
* **State Management:** React Hooks (`useState`, `useMemo`)
* **Styling:** Modern CSS (Flexbox & Grid)
* **Icons:** Custom SVG & AI-generated branding
